# BETA 199 — Mini-cartes de disciplines de l’accueil

## Correctif principal
La rangée horizontale des disciplines sur l’accueil utilisait encore un ancien rendu qui gardait :
- une icône premium trop lourde pour ce petit format,
- un double habillage arrondi,
- un centrage visuellement mauvais.

## Ce qui change
- une seule pastille d’icône dans cette zone
- icône simple, sans double carré arrondi
- centrage corrigé
- layout plus propre pour les mini-cartes de disciplines

## But
Supprimer l’effet moche de carré dans un carré et rendre la rangée plus nette, plus lisible et plus cohérente avec le reste.
