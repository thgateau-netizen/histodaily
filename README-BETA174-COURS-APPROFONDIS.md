# HistoDaily 1.0.0-beta.174 — cours approfondis

Cette version conserve le moteur stabilisé de la beta 172 et les quiz de la beta 173. La passe beta 174 porte sur la qualité pédagogique des cours.

## Travail réalisé

- Audit des 67 cours publiés.
- Enrichissement ciblé des 41 cours dont la partie complète était nettement plus courte.
- Ajout de 123 sections spécifiques : exemples documentés, acteurs, institutions, sources, mécanismes et conséquences.
- Longueur minimale d’un cours complet portée d’environ 960 à plus de 1 900 caractères.
- Longueur médiane portée à environ 2 340 caractères.
- Conservation des 26 cours déjà suffisamment développés afin de ne pas les alourdir artificiellement.
- Aucun changement du moteur, des scores, de la navigation ou du fonctionnement des quiz.

## Principes éditoriaux

Chaque cours retravaillé doit maintenant :

1. expliquer un mécanisme plutôt que seulement énumérer des faits ;
2. donner des exemples précis et nommer les acteurs ou institutions utiles ;
3. distinguer les traces disponibles de l’interprétation historique ;
4. signaler les simplifications et les idées reçues ;
5. relier le sujet à ses effets politiques, sociaux, économiques ou culturels.

## Contrôles

- 67 cours chargés ;
- 41 cours enrichis ;
- 123 nouvelles sections ;
- aucun cours complet sous 1 900 caractères ;
- cinq questions par cours conservées ;
- syntaxe JavaScript validée ;
- cache PWA isolé pour la beta 174.
