# BETA 195 — Cadrage et taille des illustrations

## But
Corriger deux défauts visibles :
- les illustrations étaient parfois trop petites
- certains cadrages donnaient une impression de décalage ou de découpe maladroite

## Modifications
- illustrations réintroduites et agrandies dans les cartes de mode
- visuels agrandis dans les cartes de disciplines
- visuels agrandis dans les cartes de recommandations
- nouveaux conteneurs plus stables avec bord arrondi, fond discret et meilleur recadrage
- grossissement léger des SVG pour mieux occuper l'espace sans changer leur légèreté

## Résultat attendu
- les images se voient vraiment
- les cartes paraissent plus éditoriales
- moins d'effet "petit pictogramme perdu"
- moins de décalage visuel sur mobile
