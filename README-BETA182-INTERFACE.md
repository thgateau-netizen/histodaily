# HistoDaily beta 182 — interface allégée

Cette version ne change ni la progression, ni les scores, ni les sauvegardes. Elle réorganise l’interface autour de l’action utile.

## Accueil
- suppression de la grosse carte de progression répétée et des boutons Parcours / Classement / Profil déjà présents dans la navigation basse ;
- programme quotidien compact sur trois colonnes ;
- carte « À continuer » ramenée à l’essentiel ;
- recommandations sous forme de carrousel court ;
- notes de version repliables.

## Parcours
- sélecteur de disciplines horizontal et compact ;
- progression guidée raccourcie ;
- cartes de chapitres rendues immédiatement pour éviter les grands blocs vides observés sur certaines captures mobiles ;
- hiérarchie plus nette entre discipline, prochaine étape et chapitres.

## Profil
- progression et collections restent visibles ;
- jeton culturel, social et réglages sont séparés dans des volets repliables ;
- les événements déjà associés aux boutons sont conservés lors du rangement des cartes.

## Fluidité
- ombres et flous réduits en mode fluide ;
- transitions courtes ;
- respect de `prefers-reduced-motion` ;
- zones horizontales avec défilement natif et sans lourde animation.
