# HistoDaily beta 179 — progression renforcée

## Nouveautés

- Collections : une médaille se débloque quand tous les cours d’un thème sont validés.
- Cours connexes : trois suggestions apparaissent à la fin d’un cours validé.
- Maîtrise par domaine : progression explicite et pénalité temporaire pour les erreurs non révisées.
- Quiz de synthèse : un bilan de 10 questions se débloque après chaque tranche de 10 cours validés dans une discipline. Validation à 8/10, récompense unique de 100 XP.
- Révisions intelligentes : chaque erreur rejoint une file dédiée et disparaît après une bonne réponse. Chaque correction rapporte 5 XP.

Les anciennes sauvegardes restent compatibles : les nouvelles données sont ajoutées à l’état existant sans réinitialisation.
