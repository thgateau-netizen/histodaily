# HistoDaily beta 188 — Expédition guidée, accès totalement libre

## Principe

L’accueil propose une expédition quotidienne cohérente : résoudre, comprendre, relier et retenir. Cette expédition n’est jamais obligatoire.

## Accès libre

- Tous les cours restent disponibles dans le catalogue.
- Une recherche globale ouvre directement n’importe quel cours.
- Les saisons sont seulement des sélections éditoriales.
- La carte du savoir est une navigation supplémentaire, pas un verrou.
- Le cours lié au mystère du jour n’est plus bloqué. Une alerte de spoil laisse le choix au joueur.

## Nouveaux systèmes

- Expédition quotidienne en quatre étapes.
- Recherche globale dans toutes les disciplines.
- Carte du savoir par domaine et notions.
- Trois saisons éditoriales construites à partir des cours existants.
- Profil de curiosité calculé à partir des cours, quiz et révisions.

## Compatibilité

Les sauvegardes précédentes restent compatibles. Aucun cours, score, mystère ou système de progression n’est supprimé.
