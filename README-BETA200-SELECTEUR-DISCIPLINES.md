# BETA 200 — Sélecteur de disciplines réellement corrigé

## Cause des échecs précédents
Les captures montraient le sélecteur horizontal de l'accueil (`home-mode-switcher`). Les versions 197 et 199 avaient principalement modifié d'autres composants proches, notamment la carte du savoir, sans remplacer le rendu réellement actif sur l'accueil.

## Correctif beta 200
- modification du HTML actif du sélecteur de disciplines de l'accueil
- modification du sélecteur équivalent dans la page Cours
- suppression de l'icône premium imbriquée dans une seconde pastille
- une seule pastille circulaire par discipline
- centrage forcé de l'icône et suppression des fonds / bordures internes
- nouveaux pictogrammes pour Musique, Cinéma et Économie

## Notes de version
Les notes visibles dans l'application ont été réécrites et ne parlent plus du correctif classement de la version précédente.

## Cache
La version, les paramètres d'assets et le cache PWA sont alignés sur `1.0.0-beta.200`.
