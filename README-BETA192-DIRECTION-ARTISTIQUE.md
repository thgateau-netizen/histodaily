# BETA 192 — Direction artistique légère

## Objectif
Passer d'une interface simplement propre à une interface plus désirable, plus chaleureuse et plus premium.

## Ce qui a été ajouté
- nouveau fichier `app-art.js`
- mini-illustrations SVG par discipline : histoire, art, cinéma, science, économie, géographie, musique, astronomie
- intégration des illustrations dans :
  - cartes de disciplines
  - carte de mise en avant d'une discipline
  - carte `Continuer`
  - cartes de recommandations
  - cartes de saisons éditoriales
- remplacement du dernier bouton de recherche libre qui utilisait encore un glyphe isolé

## Pourquoi c'est mieux
- l'app garde sa légèreté
- chaque discipline commence à avoir une identité visuelle
- les cartes importantes deviennent moins froides
- le rendu fait plus produit éditorial que simple interface utilitaire

## Philosophie
Pas de grosses images raster, pas de bibliothèque lourde : uniquement des SVG inline pour garder de bonnes perfs sur mobile.
