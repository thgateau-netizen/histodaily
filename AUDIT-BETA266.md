# Beta 266 — accueil mobile compact

Corrections ciblées d'après les captures iPhone :

- la carte d'expédition ne monopolise plus presque tout l'écran ;
- les titres longs disposent de plus de largeur et d'une taille adaptée ;
- l'illustration reste présente mais ne concurrence plus le texte ;
- le parcours Problème → Cours → Révision est placé avant le bouton d'action ;
- le dock de navigation est légèrement réduit ;
- une marge basse garantit que le contenu peut défiler au-dessus du dock ;
- le correctif de série de la beta 265 est conservé ;
- aucun changement des routes sociales, de Supabase ou du calcul des scores.
